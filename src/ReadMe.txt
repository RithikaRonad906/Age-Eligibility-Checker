/*
   Age Eligibility Checker

   Description:
   This Java program checks whether a person is eligible to vote based on their age.

   How to Use:
   1. Compile the program using a Java compiler:
      ```
      javac age.java
      ```

   2. Run the program:
      ```
      java age
      ```

   3. You will be prompted to enter the age of the person.

   4. The program will display a message indicating whether the person is eligible to vote or not based on their age.

   Example:
Enter age: 25
Age 25 is eligible to vote
